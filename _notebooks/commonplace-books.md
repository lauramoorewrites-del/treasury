# Commonplace Books

The original Greek term was τόπος κοινός (*topos koinos*), translated into Latin as *locus communis* — literally a "common place." A "place" here was not a geographical location but a mental location, a slot in an organized memory system. Rhetoricians such as Aristotle and Quintilian taught that invention — finding arguments — depended on having a set of such places available in memory.

The modern sense of "commonplace" as *ordinary* is a later degradation. The original meaning was closer to *generally useful* — a heading under which you'd file material deployable across many occasions.
